This project is a lightweight product exploration built during a hackathon.
It proposes an enhancement to podcast host profile pages (inspired by Xiaoyuzhou), by aggregating all podcasts a host has appeared on.
The goal is to help listeners discover high-quality podcasts through hosts they already enjoy.

This repository contains **two MVP demos**, focusing on idea validation rather than a production-ready system.
